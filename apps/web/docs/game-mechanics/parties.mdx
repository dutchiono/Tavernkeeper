# Parties

Parties are groups of up to 5 heroes that explore dungeons together. The party system enables cooperative gameplay and strategic team composition.

## Party Basics

### Composition
- **Minimum**: 1 hero (solo runs are possible)
- **Maximum**: 5 heroes
- **Members**: Can include your own heroes and heroes from other players

### Party Status
Parties have different statuses:
- **waiting**: Party is formed but hasn't started a run
- **ready**: Party has enough members and is ready to start
- **in_progress**: Party is currently in a dungeon run
- **completed**: Party finished a dungeon run
- **cancelled**: Party was cancelled

## Creating a Party

### Step 1: Create the Party
1. Navigate to the Party page
2. Click "Create Party"
3. Select which of your heroes to include
4. Optionally select a target dungeon

### Step 2: Add Members
You can add more heroes to your party:
- Add your own heroes directly
- Invite other players to join with their heroes

### Step 3: Invite Others
To invite other players:
1. Generate an invite code from your party page
2. Share the code with friends
3. They can join using the code

## Party Invites

### Invite Codes
- Unique codes generated for each party
- Can have expiration dates
- Can have usage limits (default: 10 uses)
- One-time use per person

### Joining a Party
1. Enter the invite code
2. Select which hero to bring
3. Join the party

## Starting Dungeon Runs

### Manual Start
The party leader can manually start a dungeon run:
1. Select a dungeon
2. Click "Start Run"
3. The run begins with all party members

### Auto-Start
When a party reaches 5 members and has a dungeon selected, the run may automatically start.

## Party Coordination

### During Runs
- Heroes make autonomous decisions via AI agents
- Party members can coordinate through the game interface
- Shared objectives must be completed
- Loot is distributed among party members

### Communication
- In-game chat and coordination
- Party status updates
- Real-time run progress

## Party Benefits

### Advantages
- **Strength in Numbers**: More heroes = better combat effectiveness
- **Role Diversity**: Mix different hero types for balanced parties
- **Shared Rewards**: Loot and experience distributed among members
- **Social Play**: Play with friends and community members

### Strategy
Consider party composition:
- **Tank**: High CON hero to absorb damage
- **DPS**: High STR/INT heroes for damage
- **Support**: High WIS/CHA heroes for healing and buffs
- **Utility**: High DEX heroes for exploration and traps

## Leaving Parties

Players can leave parties at any time:
- Before a run starts: Simply leave
- During a run: May affect run outcome
- After a run: Leave to join another party
