# AI Agents

TavernKeeper heroes are powered by ElizaOS agents that make autonomous decisions during dungeon runs. Agents have personalities, memories, and can learn from experiences.

## ElizaOS Integration

### What is ElizaOS?
ElizaOS is an AI agent framework that enables:
- Autonomous decision-making
- Personality-driven behavior
- Memory and learning
- Natural language interaction

### Hero Agents
Each hero has an ElizaOS agent that:
- Makes decisions during dungeon runs
- Has a unique personality
- Remembers past experiences
- Learns and adapts

## Agent Decision-Making

### During Dungeon Runs
Agents make decisions about:
- **Movement**: Where to explore
- **Combat**: How to fight (attack, defend, use items)
- **Exploration**: What to investigate
- **Coordination**: How to work with party members

### Decision Factors
Decisions are based on:
- Hero personality
- Current situation
- Party composition
- Past experiences (memory)
- Available actions

## Personality System

### Personality Traits
Agents have personality traits that influence decisions:
- **Aggressive**: Prefers combat and direct action
- **Cautious**: Prefers defense and careful exploration
- **Curious**: Prefers exploration and investigation
- **Social**: Prefers coordination and teamwork

### Personality Expression
Personalities affect:
- Combat style
- Exploration behavior
- Risk tolerance
- Party interaction

## Memory System

### What is Remembered
Agents remember:
- Past dungeon runs
- Successful strategies
- Failed attempts
- Party interactions
- Important events

### Memory Usage
Memories influence:
- Future decisions
- Strategy selection
- Risk assessment
- Party coordination

## Agent Actions

### Available Actions
Agents can choose from:
- **Move**: Navigate the dungeon
- **Attack**: Engage in combat
- **Defend**: Take defensive stance
- **Use Item**: Consume items or use equipment
- **Skill Check**: Attempt skill-based actions
- **Interact**: Interact with objects or NPCs

### Action Selection
Agents select actions based on:
- Current situation
- Personality
- Memory
- Available options

## Agent Communication

### In-Game Communication
Agents can communicate:
- With party members
- About strategy
- About discoveries
- About threats

### Natural Language
Agents use natural language for:
- Decision explanations
- Party coordination
- Situation assessment
- Social interaction

## Agent Learning

### Experience-Based Learning
Agents learn from:
- Successful strategies
- Failed attempts
- Party feedback
- Outcome analysis

### Adaptation
Agents adapt their behavior:
- Based on what works
- Based on party composition
- Based on dungeon type
- Based on past experiences

## Agent Configuration

### Initial Setup
Agents are configured with:
- Personality traits
- Initial memories
- Decision preferences
- Communication style

### Customization
Players can influence:
- Agent personality (to some extent)
- Initial memories
- Decision priorities
- Communication preferences
