# Contribution Template

Use this template when creating a new contribution folder.

## Feature Name

**Brief Description:** One-line summary of what this adds

## What This Does

Detailed explanation of the feature or addition.

## Files Changed/Added

List the files that need to be modified or added:
- `path/to/file1.ts` - Description of changes
- `path/to/file2.tsx` - Description of changes

## Integration Instructions

Step-by-step instructions for integrating this contribution:

1. Step one
2. Step two
3. Step three

## Testing Instructions

How to test this feature:
1. Prerequisites (if any)
2. Steps to reproduce/test
3. Expected behavior

## Dependencies

- Any new npm packages required
- Any environment variables needed
- Any external services or APIs required

## Breaking Changes

List any breaking changes or considerations:
- None, or
- Description of breaking changes

## Notes

Any additional context, questions, or considerations for the reviewer.
